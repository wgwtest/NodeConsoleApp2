地图包导入导出与校验

这是 WBS-3.2.3.3 的独立 IO 校核页。它不负责拖拽地图节点，也不重复承担 WBS-3.2.3.2 的编辑职责；它只负责验证地图包是否符合 JSON + image assets 契约，能否被导入、规范化导出、发现非法字段，并执行 round-trip 稳定性检查。

本页用途

这是个啥工具

它是地图包的“结构校核页”。你在这里验证的是 JSON 契约和图片资源引用，不是地图舞台视觉，不是战斗主流程，也不是关卡编辑器本身。

为什么点这些按钮

Load 合法样例 用来确认正常地图包可以稳定导入；Load 非法样例 用来确认问题码不是靠猜；Run round-trip 用来确认导入后再导出不会把结构悄悄弄坏。

与 3.2.3.2 的关系

level_map_editor_v1.html 是作者编辑页，负责节点、边、背景的操作；本页负责证明这些产物进入地图包后，导入导出与字段级校验是稳定的。

你该看什么结论

先看“合法样例 / 非法样例 / round-trip”三块结论，再看问题码列表。只要问题码能稳定复现、合法样例能稳定归零，就说明 `WBS-3.2.3.3` 的工具边界是清楚的。
