这个地图包是什么？

当前导出的是完整目录式地图包：package.json 是包入口，maps.json 保存故事、章节、地图、节点、边和显示配置，asset-manifest.json 列出背景图、节点素材图和立绘图的资源 ID / 路径引用。图片文件不会被嵌入 JSON。导出只负责下载备份；保存工作稿和发布到主流程使用顶部按钮。
