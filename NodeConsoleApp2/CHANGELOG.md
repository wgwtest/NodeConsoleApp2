# 版本说明与更新日志

最后整理时间：2026-04-14

状态：`当前有效`

## 当前版本定位

当前版本不是打包分发版，而是一个可供继续研发、回归和人工验收的候选版。

它当前覆盖：

1. `mock_ui_v11.html` 的主流程游玩入口
2. `test/codex_regression_runner.html` 的共享回归入口
3. 技能 / Buff / 关卡 / 地图 / 表现层的专项工具与 probe 页
4. `DOC/CODEX_DOC/` 下的设计、计划、测试与交接证据链

## [未发布]

### 新增

1. `WBS-3.3.3.1` 已把旧 `skill tags` 从正式功能、契约校验、工具链和活动技能包中退役，改为只依赖正式字段与派生视图。
2. `WBS-4.2.1 ~ WBS-4.2.3` 已在主流程页补齐页面用途说明、机制术语卡和结构化阻塞反馈。
3. `WBS-5.1` 已建立发布候选版的测试入口分层：
   - 正式主流程入口
   - 共享回归入口
   - 人工验收入口
4. `WBS-5.2` 已固定当前运行方式、交付目录口径和版本边界说明。
5. 新增 `npm run test:acceptance-clicks` 自动点击验收烟测，覆盖主流程、展示层 probe 与展示配置器 JSON/workspace 往返。

### 变更

1. `package.json` 不再保留空壳测试脚本，改为暴露可直接运行的服务与回归命令。
2. 仓库根 `README.md` 改为以当前真实工程入口、启动命令和正式文档根为主，而不是历史原型说明。
3. 验收总入口与测试文档索引改为候选版固定入口，不再只是一份临时导航说明。

### 已知问题

1. `#73 / WBS-4.3.2.1` 的人工验收清单仍需重写，当前不适合直接交给验收人点击。
2. `#82 / WBS-4.3.5` 的人工验收清单仍需重写，当前边界说明不够清楚。
3. 当前工程仍依赖浏览器 `localStorage` 承接部分 runtime override、测试工作区和本地存档；如果不清理旧数据，可能影响复测结果。
4. 当前仓库不负责 Spine 素材制作工程本体；该部分协作已拆到独立仓库和独立工作流。

### 交付目录

当前候选版的主要交付目录如下：

1. `mock_ui_v11.html`
   - 正式主流程入口
2. `test/`
   - 回归页、probe 页、工具页和节点级测试
3. `assets/data/`
   - 当前活动技能 / Buff / 关卡 / 地图数据
4. `script/`
   - 运行时与工具链代码
5. `DOC/CODEX_DOC/`
   - 正式本地文档根

### 不包含内容

以下内容当前不属于本仓库交付承诺：

1. 安装包、升级器或桌面发布物
2. 独立的 Spine 素材制作工程源码
3. 一切未进入 WBS 执行树的探索性原型
