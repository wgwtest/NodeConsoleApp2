# 敌人原画生成提示词 v2

用途：把本文内容复制到图片生成工具中，配合上传的参考图生成敌人战斗完整原画。本文不依赖任何本地路径；生成时请优先以上传的参考图片作为风格、画布比例、主体大小和脚底位置参考。

## 使用方式

1. 上传当前项目里的基准敌人原画作为参考图。
2. 复制“通用母提示词”。
3. 再追加一个“单个敌人提示词”。
4. 同时追加“通用负面提示词”。
5. 生成 PNG 透明底完整角色原画。

## 基准参考要求

以上传的参考图为准，目标约束如下：

```text
画布尺寸：1021 x 1062 px
文件格式：PNG
颜色模式：RGBA
背景：透明 alpha
角色必须是完整站姿全身，不是头像、半身、胸像或地图节点图标
角色可见高度：约 860 到 890 px
角色可见宽度：普通敌人约 760 到 900 px；长弓、盾牌、长柄武器可接近 980 px，但不能裁切
脚底基线：接近画布底部，约 y=1018 到 y=1062
角色整体居中，身体朝向画面左侧三分之二侧身，面部、鼻尖和视线也必须朝向画面左侧
```

重点：不要让新角色比上传参考图明显更小；不要把角色缩在画布中央；不要让脚底悬空。

## 生成后人工检查

```text
1. 是否为 PNG + RGBA 透明底。
2. 画布是否为 1021 x 1062 px。
3. 是否没有白底、黑底、灰底、场景底图或地面。
4. 是否完整全身，头、脚、武器、盾牌、弓、披风没有被裁切。
5. 主体是否接近上传参考图的大小；如果明显小一圈，需要重新生成或放大。
6. 脚底是否接近画布底部；如果悬在中间，会在战斗界面里显得漂浮。
7. 面部、鼻尖和视线是否朝向画面左侧；如果面朝右侧或正面直视镜头，放进敌方右侧位置会显得方向错误。
8. 同类敌人是否能从轮廓上区分：哥布林小而狡猾，兽人高大厚重，骷髅必须有骨架特征。
```
