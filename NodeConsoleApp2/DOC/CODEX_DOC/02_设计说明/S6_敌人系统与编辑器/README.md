## 3. 归属规则

1. 敌人模板字段、种族、职业、属性、部位护甲、技能组和 AI 行为语义归入本子系统。
2. 敌人编辑器归入本子系统，作为敌人模板内容生产工具。
3. 敌人技能定义归入 `S4` 的 skill schema 与技能编辑器能力；敌人模板只保存 `skills[]` 引用和 AI 语义。
4. 关卡编辑器与地图编辑器只消费敌人模板目录、摘要和 `templateId`，不直接维护敌人模板真值。
5. 战斗运行时如何实例化敌人、推进回合和执行行动归入 `S2_战斗运行时`；本子系统定义敌人内容语义与作者工具边界。
6. 跨模块数据装载、存档兼容和通用内容契约归入 `S7_数据存档与内容契约`。
