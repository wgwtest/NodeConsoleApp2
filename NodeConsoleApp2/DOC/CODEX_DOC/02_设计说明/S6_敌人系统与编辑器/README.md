# S6 敌人系统与编辑器设计说明

最后整理时间：2026-05-23

状态：`当前有效`

## 1. 子系统范围

本子系统负责敌人模板、敌人种族与职业、敌人属性与部位护甲、敌人技能组、敌人 AI 设计语义、敌人模板编辑器，以及与关卡编辑器、地图编辑器之间的 `templateId` 交接边界。

## 3. 归属规则

1. 敌人模板字段、种族、职业、属性、部位护甲、技能组和 AI 行为语义归入本子系统。
2. 敌人编辑器归入本子系统，作为敌人模板内容生产工具。
3. 关卡编辑器与地图编辑器只消费敌人模板目录、摘要和 `templateId`，不直接维护敌人模板真值。
4. 战斗运行时如何实例化敌人、推进回合和执行行动归入 `S2_战斗运行时`；本子系统定义敌人内容语义与作者工具边界。
5. 跨模块数据装载、存档兼容和通用内容契约归入 `S7_数据存档与内容契约`。
