## 3. 归属规则

1. 跨子系统共享数据、资产包字段、存档 schema、装载兼容归入本子系统。
2. 单个编辑器的 UI 字段布局归入对应编辑器子系统。
3. 单个运行时行为归入对应运行时子系统。
