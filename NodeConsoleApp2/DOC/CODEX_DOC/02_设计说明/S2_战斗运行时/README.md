## 3. 归属规则

1. 运行时行为、回合推进、战斗事件、敌人实例化消费边界、道具效果归入本子系统。
2. 技能和 Buff 的配置来源分别归入 `S4` 和 `S5`，本子系统只描述消费边界。
3. 敌人模板、敌人 AI 语义和敌人编辑器归入 `S6`；本子系统只描述战斗运行时如何消费敌人模板。
4. 纯 UI 布局和交互规范归入 `S8`。
