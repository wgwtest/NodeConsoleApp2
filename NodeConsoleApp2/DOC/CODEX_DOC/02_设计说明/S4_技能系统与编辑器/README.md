## 4. 归属规则

1. 技能数据、技能树节点、技能编辑器字段、技能运行时说明归入本子系统。
2. Buff action 与 Buff 参数契约归入 `S5`；技能文档只描述如何引用和消费 Buff。
3. 敌人技能包使用同一套技能 schema，但其敌人模板消费、敌人 AI 语义和敌人编辑器引用边界同时关联 `S6`。
4. 技能树 UI 原型放入 `08_原型与附图/`。
5. 技能树页面的统一视觉与模态交互规则归入 `S7`。
