# S1 主游戏流程设计说明

最后整理时间：2026-05-21

状态：`当前有效`

## 1. 子系统范围

本子系统负责玩家从主入口进入游戏、查看状态、选择关卡、确认进入、完成结算并回到成长或下一局流程的整体路径。

## 3. 归属规则

1. 主菜单、关卡选择、进入确认、结算跳转、流程状态摘要归入本子系统。
2. 地图数据、地图节点、地图编辑器归入 `S3_关卡地图与编辑器`。
3. 技能树、技能学习、构筑摘要归入 `S4_技能系统与编辑器`。
4. 视觉样式和统一交互规则归入 `S8_UI与交互基线`，本子系统只保留流程语义。
