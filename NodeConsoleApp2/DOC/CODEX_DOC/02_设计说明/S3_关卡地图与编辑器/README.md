1. 地图节点、地图切换、地图缩放拖拽、地图包导入导出归入本子系统。
2. 关卡进入确认和主流程跳转归入 `S1`。
3. 关卡运行时战斗执行归入 `S2`。
4. 地图视觉原型放入 `08_原型与附图/`，本子系统只引用正式原型路径。
