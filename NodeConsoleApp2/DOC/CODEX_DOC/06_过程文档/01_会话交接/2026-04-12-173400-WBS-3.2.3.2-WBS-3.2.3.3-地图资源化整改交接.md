# 2026-04-12 WBS-3.2.3.2 / WBS-3.2.3.3 地图资源化整改交接

## 1. 本轮目标

处理远端评论指出的两类失败：

1. `WBS-3.2.3.2`
   - 页面丑、布局乱、文本溢出
   - 背景图和节点图片配置缺失
   - “素材化地图舞台”实际只是程序生成占位图
2. `WBS-3.2.3.3`
   - 地图包 IO 页没有把 `JSON + image assets` 方案讲清楚
   - 非法样例未覆盖图片资源引用缺失

## 2. 根因判断

本轮不是“少改了几个样式”，而是方案偏航：

1. 数据字段上虽然已有 `backgroundRef / artRefs.*`
2. 但页面实际消费的是 data-url 动态占位图，而不是真实图片资源
3. 自测只验证了结构与交互，没有验证“真实资源是否被消费”

4. 编辑页、预览页、IO 页共用这套资源解释口径
