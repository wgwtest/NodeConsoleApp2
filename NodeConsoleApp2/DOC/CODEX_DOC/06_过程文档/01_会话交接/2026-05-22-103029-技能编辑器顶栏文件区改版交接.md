# 技能编辑器顶栏文件区改版交接

创建时间：2026-05-22 10:30:29 +0800

## 1. 本轮处理内容

本轮基于已确认的顶栏原型实施运行页改版，只处理技能编辑器顶部文件和操作区。

核心变化：

1. `加载版本` 改为 `打开版本`。
2. `加载项目` 改为 `打开路径`。
3. `导入本地` 改为 `从本地导入`。
4. `加载 Buff` 改为 `手动加载 Buff`，并明确展示 `同目录自动加载`。
5. `另存为` 改为 `另存到指定路径`。
6. `下载` 改为 `下载备份`。

## 2. 关键设计判断

`加载项目` 和 `加载版本` 的差别不应该靠用户猜。现在按来源拆分：

1. 最近版本：从后端扫描出的技能 JSON 列表中打开。
2. 指定路径：从项目内指定 JSON 路径打开。
3. 从本地导入：从浏览器文件选择器读取本机 JSON，只作为导入来源。

保存发布区只强调两个主流程动作：

1. 保存工作稿。
2. 发布到主流程。

备份类动作降级到第二行。
