## 本轮调整

### 敌人技能

- `enemy_skill_bad_lunchbox`
  - 伤害 `4 -> 8`
  - 中毒概率 `0.65 -> 0.85`
  - 目的：让支援 / 状态敌人的毒性骚扰转化成可见生命损耗。
- `enemy_skill_dirty_bandage`
  - 治疗 `14 -> 10`
  - 目的：降低自我治疗长期压过攻击选择的倾向。
- `enemy_skill_shield_tap`
  - 保留原有 `DMG_ARMOR 12`。
  - 新增 `DMG_HP 8` 的头部冲击动作。
  - 目的：让护甲敌人破甲时也能制造生命压力，而不是只打护甲。
- `enemy_skill_bone_patch`
  - 补甲 `12 -> 8`
  - 目的：让修复节奏不再把战斗拖成低风险循环。
- `enemy_skill_dead_air`
  - 伤害 `5 -> 8`
  - 减速概率 `0.75 -> 0.85`
  - 目的：加强腿部骚扰和速度压力。

### 诊断口径

- `tools/campaign_balance_simulator.mjs`
  - `enemy_too_weak` 从固定 `playerRemainingHp >= 90 && turns <= 3` 改为 `playerRemainingHp / playerMaxHp > 0.9 && turns <= 3`。
  - 原因：recommended 构筑当前 `playerMaxHp = 120`，使用固定 90 HP 阈值会把 `104/120` 这类已有损耗的胜利误判为过弱。
