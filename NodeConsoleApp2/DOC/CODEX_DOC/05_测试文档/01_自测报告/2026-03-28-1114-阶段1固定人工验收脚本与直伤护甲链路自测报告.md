## 2. 本轮关键结论

### 2.1 新发现并修复的真实运行时缺口

在真实页面复测 `护甲免伤演示` 时，发现原始样本会出现：

1. 护甲不掉
2. 但 HP 仍下降

继续定位后确认问题不在 Buff 数据，而在运行时：

1. `DMG_ARMOR` 旧分支直接调用 `_applyArmorDelta()`
2. 没有经过 `BATTLE_TAKE_DAMAGE_PRE`
3. `PREVENT_DAMAGE_ARMOR` 与 `HEAL_ARMOR` 因此不会在直伤护甲技能上生效

本轮已修复：

1. 新增直伤护甲专用战斗管线 `_applyArmorOnlyDamage()`
2. `DMG_ARMOR` 现在会经过 `BATTLE_ATTACK_PRE -> BATTLE_TAKE_DAMAGE_PRE -> BATTLE_TAKE_DAMAGE -> BATTLE_ATTACK_POST -> BATTLE_DEFEND_POST`
3. 同时保持“只伤护甲，不穿 HP”的语义

### 2.2 验收样本已拆成固定场景

当前真实页面已具备以下专用验收关卡：

1. `人工验收样本`
   - 用于 `HP免伤`
2. `护甲免伤验收样本`
   - 使用固定头部小额护甲攻击敌人
3. `受击获甲验收样本`
   - 开战玩家头部护甲为 `0 / 20`
4. `敌人修甲验收样本`
   - 敌人胸部开战即为 `4 / 24`
5. `敌人回血验收样本`
   - 敌人开战 HP 为 `18 / 90`
