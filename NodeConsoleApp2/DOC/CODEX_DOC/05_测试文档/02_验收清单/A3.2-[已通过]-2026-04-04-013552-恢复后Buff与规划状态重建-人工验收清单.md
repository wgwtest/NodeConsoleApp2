## 1. 验收目标

确认恢复战斗后，运行时状态会被正确重建，而不是只回到空壳战斗界面。

## 2. 验收入口

1. `mock_ui_v11.html`
2. 基于 `A3.1` 已完成一次保存

## 3. 操作步骤

1. 完成 `A3.1` 的全部步骤并保持存档存在
2. 刷新 `mock_ui_v11.html`
3. 再次点击 `开始冒险`
4. 在主菜单点击 `继续游戏`

## 4. 通过标准

恢复后界面应同时满足：

1. 回合数仍显示为 `1`
2. 时间线仍显示 2 条待执行动作
3. 时间线内容与保存前一致：
   - 敌方 `skill_bite`
   - 我方 `skill_heavy_swing`
4. 页面无明显报错或空白

## 5. 失败表现

任一情况均判未通过：

1. 恢复后回合数被重置
2. 时间线为空
3. 时间线内容与保存前不一致
4. 无法进入当前战斗，只能重新选关
