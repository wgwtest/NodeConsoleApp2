## 1. 验收目标

确认当前项目已经具备一条可复现的结果与奖励结算闭环：

1. 自然胜利 / 自然失败都能从真实执行链进入结算态。
2. 结算时会展示真实奖励，而不是固定文案。
3. 奖励会写入玩家数据与存档。
4. 确认结算后可以返回主菜单。

## 2. 验收入口

1. `test/codex_regression_runner.html`
2. `mock_ui_v11.html`
3. 浏览器控制台

## 3. 前置条件

1. 使用本地静态服务打开页面。
2. 为避免旧存档干扰，验收前先在控制台执行：

```js
localStorage.removeItem('save_game');
location.reload();
```

## 4. 操作步骤

### 4.1 回归补充佐证

1. 打开 `http://127.0.0.1:3101/test/codex_regression_runner.html`
2. 确认以下条目为 `PASS`
   - `DataManager 结算奖励会入账到玩家与进度`
   - `CoreEngine.endBattle 会发出带 settlement 的结算事件`
   - `CoreEngine 在战斗开始时应用关卡玩家战斗数值覆盖`
   - `CoreEngine 自然执行链会触发胜利结算`
   - `CoreEngine 自然执行链会触发失败结算`
   - `UI_SystemModal 会渲染奖励摘要`
3. 确认既有回归没有新增失败

### 4.2 主页面结算展示与入账验证

1. 打开 `http://127.0.0.1:3101/mock_ui_v11.html`
2. 点击 `开始冒险`
3. 在菜单中点击 `关卡选择`
4. 选择 `幽暗森林边缘`
5. 等待页面进入战斗规划态
6. 在浏览器控制台执行以下脚本，记录结算前资源：

```js
({
  kp: window.GameEngine.data.playerData.skills.skillPoints,
  resources: window.GameEngine.data.playerData.resources
})
```

7. 在控制台执行：

```js
window.GameEngine.endBattle(true);
```

8. 观察点：
   - 模态窗标题为 `战斗胜利`
   - 弹窗正文出现 `关卡：幽暗森林边缘`
   - 弹窗正文出现：
     - `经验 +100 / 累计 100`
     - `金币 +50 / 累计 50`
     - `知识点 KP +1 / 累计 31`
   - 弹窗出现 `首次通关`
9. 点击 `返回主菜单`
10. 在控制台执行：

```js
({
  kp: window.GameEngine.data.playerData.skills.skillPoints,
  resources: window.GameEngine.data.playerData.resources,
  save: JSON.parse(localStorage.getItem('save_game'))
})
```

11. 观察点：
   - `kp` 比结算前增加 `1`
   - `resources.exp === 100`
   - `resources.gold === 50`
   - `save.runtime.currentScene === 'MAIN_MENU'`
   - `save.global.progress.completedLevels` 包含 `level_1_1`
   - `save.global.progress.lastSettlement.levelId === 'level_1_1'`

## 5. 通过标准

满足以下全部条件才算通过：

1. 回归页中 6 条 B1 相关条目保持 `PASS`
2. 自然胜利 / 自然失败已由回归页验证可进入结算态
3. 结算弹窗中显示真实奖励摘要，而不是只有固定胜利文案
4. 奖励会写入玩家 `KP / exp / gold`
5. 返回主菜单后，`save_game` 中保留本次 settlement 与 completedLevels

## 6. 失败表现

任一情况均判未通过：

1. 弹窗没有显示关卡名或奖励数值
2. 回归页中的自然胜利 / 自然失败任一条未通过
3. 结算后 `KP / exp / gold` 未变化
4. `save_game.runtime.currentScene` 不是 `MAIN_MENU`
5. `lastSettlement` 或 `completedLevels` 丢失
