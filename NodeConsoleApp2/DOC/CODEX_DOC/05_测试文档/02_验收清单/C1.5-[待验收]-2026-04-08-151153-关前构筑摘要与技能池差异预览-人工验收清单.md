# C1.5 关前构筑摘要与技能池差异预览人工验收清单

对应 WBS：`#75 / C1.5 关前构筑摘要与技能池差异预览`

## 1. 本清单在验什么

只验证 `C1.5`：

1. 关卡选择页会出现 `关前构筑摘要`。
2. 该摘要会明确说明当前版本没有单独的预装配步骤。
3. 该摘要会展示当前技能池和最近学习带来的技能池差异。
4. 学习后重新进入关卡选择页，摘要会立即更新。

## 2. 验收入口

1. 主入口：`http://127.0.0.1:3101/mock_ui_v11.html`
2. 自动回归辅助入口：`http://127.0.0.1:3101/test/codex_regression_runner.html`

## 3. 前置条件

1. 使用 `http://127.0.0.1:3101/...` 打开页面。
2. 建议从能正常进入主菜单的新局或存档开始。
3. 如果主菜单残留旧局状态，先刷新页面。

## 4. 为什么这些步骤能证明 C1.5

1. `C1.5` 不是新增一套装备系统，而是把当前版本“已学技能会自动带入本局”的真实规则在关前解释清楚。
2. 所以必须同时观察：
   - 当前技能池
   - 没有单独预装配步骤的说明
   - 最近学习带来的新增差异
   - 学习后是否会即时刷新

## 5. 人工验收步骤

### 5.1 读取关前构筑摘要

步骤：

1. 打开 `mock_ui_v11.html`
2. 点击 `开始冒险`
3. 在主菜单点击 `关卡选择`

观察点：

1. 关卡选择页顶部出现 `关前构筑摘要`
2. 摘要中存在 `当前技能池`
3. 摘要中存在 `预装配说明`

### 5.2 验证当前没有单独预装配步骤

观察点：

1. 文案明确说明：
   - `本版本无单独预装配步骤`
   - 或同等含义的说明
2. 能理解当前规则是“进入关卡后自动带入全部已学技能”

### 5.3 验证技能池差异预览

观察点：

1. 摘要里能看到至少一个当前技能池样本技能名，或明确的技能池数量
2. 摘要里存在 `最近学习带来的技能池差异`
3. 如果当前已有最近学习记录，应能看到具体技能名或明确差异说明

### 5.4 学习后重新进入关前页

步骤：

1. 返回主菜单
2. 点击 `技能树 / 构筑`
3. 学习至少 1 个技能并点击 `提交并关闭`
4. 再次点击 `关卡选择`

观察点：

1. `关前构筑摘要` 仍然存在
2. 当前技能池数量或样本技能名发生更新
3. `最近学习带来的技能池差异` 同步更新

## 6. 通过标准

以下全部满足才算通过：

1. 关卡选择页出现 `关前构筑摘要`
2. 摘要明确解释当前没有单独预装配步骤
3. 摘要能展示当前技能池
4. 摘要能展示最近学习差异
5. 学习后重新进入关卡选择页会同步刷新

## 7. 自动回归辅助核对

回归页应看到以下 2 项为 `PASS`：

1. `UI_SystemModal 会在关卡选择页展示关前构筑摘要与技能池差异预览`
2. `UI_SystemModal 会在 PLAYER_SKILLS 更新后刷新关前构筑摘要`
