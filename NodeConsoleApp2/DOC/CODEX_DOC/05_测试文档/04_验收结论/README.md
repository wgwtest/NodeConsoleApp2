# 验收结论 README

最后整理时间：2026-04-03

状态：`当前有效`

## 1. 目录用途

本目录用于承接“需要从多份验收清单、用户批注或阶段审定中提炼出的最终结论文档”。

## 2. 什么时候写到这里

推荐写入本目录的情况：

1. 某个阶段需要单独形成汇总性验收结论
2. 某个节点的通过/不通过结论需要独立留档
3. 单份人工验收清单不足以承载最终结论摘要

## 3. 当前工程的关系

当前工程的节点级通过状态，仍以以下材料联合判断：

1. 对应人工验收清单中的用户批注和结论
2. GitHub Issue 正文中的 `当前状态`
3. GitHub Project 中的粗粒度状态

如果未来需要做阶段性汇总或跨节点结论沉淀，再在本目录新增正式结论文档。
