# WBS-5.1.2 人工验收入口冻结与版本结论页基线

最后整理时间：2026-04-14

状态：`当前有效`

## 1. 文档用途

本文件固定当前候选版的人工作业入口、结论留痕位置和更新规则。

## 2. 当前正式人工验收入口

1. 当前唯一正式人工验收主入口：
   - `DOC/CODEX_DOC/05_测试文档/02_验收清单/00-当前待验收功能总入口.md`
2. 当前版本候选结论页：
   - `DOC/CODEX_DOC/05_测试文档/04_验收结论/01-当前版本候选结论页.md`

## 3. 固定工作流

1. 用户说“我要验收”时，先刷新总入口。
2. 用户只在对应 WBS 清单里用 `【】` 写结论。
3. 通过结论从活跃清单同步到：
   - 清单文件名状态
   - 验收总入口
   - 验收清单 README
   - GitHub Issue / Project
4. 版本候选结论页只做汇总留痕，不替代节点级清单。

## 4. 当前候选版结论页职责

1. 汇总当前待验收节点
2. 记录不可直接人工验收的节点
3. 标明当前候选版是否已经具备“可继续收口”的状态
