# 地图编辑器本关敌人面板原型 v1

本原型基于现有 `test/level_map_editor_v1.html` 复制页制作，不是重新绘制一套地图编辑器。

## 设计边界

- 保留现有地图编辑器的左右抽屉、真实地图背景、真实节点与边。
- 只在中间主舞台下方新增“本关敌人”轻量面板。
- 当前按“一个关卡基本只有一个敌人”的口径设计，不做多敌人池成员管理。
- 面板从选中节点的 `levelId` 读取关卡，再读取第一波第一个敌人模板展示摘要。
- 完整敌人深编辑后续可另设详情入口，本原型不把复杂敌人模板编辑塞进地图页。

## 文件

- `index.html`：可交互原型页。
- `01-真实地图编辑器-本关敌人面板-抽屉展开.png`：左右抽屉展开状态。
- `02-真实地图编辑器-本关敌人面板-抽屉收起.png`：左右抽屉收起状态。
