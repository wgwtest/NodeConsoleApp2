## 设计口径

- 现有技能树不推翻，新增节点挂在现有节点上。
- 口语化、搞笑命名保留，但每个节点都要能追到动作、Buff 或明确的新引擎条件。
- 玩家可以自由横跳：新增桥接节点连接护甲、重击、流血、机动、风险几组机制。
- 每关约 3 KP：新增入口多为 2-3 KP，核心桥接多为 3-4 KP，强变招约 5 KP。
