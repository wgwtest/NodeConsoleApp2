## 设计判断

- 当前正式技能约 32 个，总 KP 约 111；按每关 3 KP，完整学完约 37 关。
- 不建议做职业互斥路线。建议做“五个机制入口 + 横向桥接节点”：
  - 护甲经济
  - 重击破甲
  - 流血收益
  - 机动节奏
  - 风险代价
- 梗名可以保留，但必须让梗变成机制。例如：
  - 临期食品：强生存收益 + 中毒/减速/随机负面。
  - 提桶跑路：回血逃命 + 清空护甲或降速。
  - 光头/光脚：主动放弃部位护甲换 AP/速度。

## 待确认问题

- “自由横跳”的强度：是所有入口无前置，还是只允许少量核心入口无前置？
- “临期食品”的副作用更偏中毒、减速，还是随机负面？
- “西海岸”是否保留为正式节点；如果保留，需要补 `buff_shield` 或重写成现有 Buff 能表达的效果。
- “吸血”是回血，还是吸护甲？当前 Buff 文件中的 `buff_lifesteal` 实际是 `HEAL_ARMOR`。
