## 4. 验收标准

1. 可以导出合法地图包
2. 可以回读地图包并恢复节点、边和背景数据
3. 至少提供一个合法样例、一个非法样例与一个 round-trip 检查
4. 工具页会明确解释“它为什么不是地图编辑页，而是 IO 校核页”
5. 共享回归页可通过 `scope=WBS-3.2.3.3` 单独查看
6. 非法样例应能稳定打出 `missing_background_ref / missing_node_art_ref / missing_portrait_ref` 等图片引用问题码
7. IO 页必须明确解释地图包采用 `JSON + image assets`，并展示背景图资源、节点素材图、立绘图的真实路径
