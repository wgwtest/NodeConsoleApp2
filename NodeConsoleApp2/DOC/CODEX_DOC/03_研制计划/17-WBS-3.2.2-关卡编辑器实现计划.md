# WBS-3.2.2 关卡编辑器实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 交付一个独立于主流程的正式关卡编辑器，支持关卡节点 CRUD、关卡连接编辑、关卡内要素编辑与 JSON 导出。

**Architecture:** 采用“纯数据工作区核心 + 独立页面壳 + 运行时验证链”三段式结构。纯数据核心负责 wrapped `levels.json` 的增删改查、关系维护和校验；页面壳只负责渲染与交互；导出后通过 runtime override 与 probe 页验证不直接耦合主流程。

**Tech Stack:** 原生 HTML / CSS / JavaScript、Node `node:test`、既有 `ContentPackOverrideStore`、`DataManagerV2`
